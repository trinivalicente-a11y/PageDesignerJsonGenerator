# PageDesignerJsonGenerator
Claude Skill to generate JSON files to import to page designer.


**Since this skill requires manual loading, you'll paste the skill files into Claude at the start of each conversation.**

**Step 1** — Clone the repo
Run git clone https://github.com/trinivalicente-a11y/PageDesignerJsonGenerator.git in your terminal.

**Step 2**— Open the 3 files in a text editor
Navigate into the landing-page-json-generator 5/ folder and open SKILL.md, references/color-palettes.md, and references/section-templates.md.

**Step 3** — Start a new Claude conversation and paste all three files
At the very beginning of the conversation, paste all three file contents in a single message, each labeled with its filename (e.g. --- SKILL.md ---), and ask Claude to read them before doing anything.

**Step 4** — Wait for Claude to confirm
Claude will confirm it has read the skill and is ready. You only need to do this once per conversation.

**Step 5**— Provide your event details
Give Claude the event name, date, location, description, logo URL, brand colors, and any optional details like sections to include (speakers, FAQ, sponsors, etc.).
